USAGE GUIDE
===============================================================================

Import Order for Neo4j:
1. Import all node files first (order doesn't matter within nodes)
2. Import all relationship files (order doesn't matter within relationships)

Example Cypher Import (nodes_organization.csv):
LOAD CSV WITH HEADERS FROM 'file:///nodes_organization.csv' AS row
CREATE (o:Organization {
  name: row.name,
  type: row.type,
  founded: date(row.founded),
  headquarters: row.headquarters,
  website: row.website,
  purpose: row.purpose
})

Example Cypher Import (relationships_publishes.csv):
LOAD CSV WITH HEADERS FROM 'file:///relationships_publishes.csv' AS row
MATCH (o:Organization {name: row.source_name})
MATCH (s:Standard {name: row.target_name, version: row.target_version})
CREATE (o)-[:PUBLISHES]->(s)
